    Variables and Types

    You do not need to declare variables before using them or declare their type. Every variable in Python is an object. Variables can hold any value and are case-sensitive. This tutorial will go over a few basic types of variables.

    Variable Names

    There are a number of rules governing the naming of variables:

    1. Variable names cannot start with a number
    2. There can’t be any spaces in a variable name
    3. Reserved words cannot be used in variable names

    Numbers

    Python supports integer and floating-point numbers. To define an integer, use the following syntax:

    myinteger=7 
print(myinteger)
