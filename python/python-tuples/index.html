    Python Tuples

    Python Tuples

    A Tuple is a collection which is ordered and unchangeable. Tuples are defined with round brackets:

    myTuple = ("SQL", "Python", "R")
print(myTuple)

    Accessing Tuple Elements

    Indexing

    An element in a tuple can be accessed by referring to it’s index number, inside square brackets. The following prints the word “Python” which is the second element in the tuple:

    myTuple = ("SQL", "Python", "R")
print(myTuple[1])

    Negative Indexing

    Negative indexing starts at the end of a tuple and indexes backwards towards the beginning. The last element in the tuple is at location -1, -2 refers to the second last item etc. The following prints the string ‘R’:

    myTuple = ("SQL", "Python", "R")
print(myTuple[-1])

    Range of Indexes

    You can specify a range of indexes by specifying where to start and where to end the range. When specifying a range, the return value will be a new tuple with the specified items. Return the third, fourth, and fifth item:

    myTuple = ("SQL", "Python", "R", "orange", "kiwi", "melon", "mango")
print(myTuple[2:5])

    Note: The search will start at index 2 (included) and end at index 5 (not included). Remember that the first item has index 0.

    Range of Negative Indexes

    Specify negative indexes if you want to start the search from the end of the Tuple. This example returns the items from index -4 (included) to index -1 (excluded)

    myTuple = ("SQL", "Python", "R", "orange", "kiwi", "melon", "mango")
print(myTuple[-4:-1])

    Modifying Tuples

    Once a tuple is created it is immutable. This means you cannot change its values.

    Adding Elements to a Tuple

    Once a tuple is created, you cannot add items to it. Tuples are unchangeable. You cannot add items to a tuple:

    myTuple = ("SQL", "Python", "R")
myTuple[3] = "orange" # This will raise an error
print(myTuple)

    Removing Elements from a Tuple

    You cannot remove items in a tuple. Tuples are unchangeable, so you cannot remove items from it, but you can delete the tuple completely. The del keyword can delete the tuple completely:

    myTuple = ("SQL", "Python", "R")
del myTuple
print(myTuple) #this will raise an error because the tuple no longer exists

    Looping Through a Tuple

    You can loop through the tuple items by using a for loop. Iterate through the items and print the values:

    myTuple = ("SQL", "Python", "R")
for x in myTuple:
  print(x)

    Check if Item Exists

    To determine if a specified item is present in a tuple the ‘in’ keyword can be used. For example, to check if “SQL” is present in the tuple:

    myTuple = ("SQL", "Python", "R")
if "SQL" in myTuple:
    print("Yes, 'SQL' is present in the tuple")

    Tuple Length

    To determine how many items a tuple has, use the len() method. Print the number of items in the tuple:

    myTuple = ("SQL", "Python", "R")
print(len(myTuple))

    Create Tuple With One Item

    To create a tuple with only one item, you have add a comma after the item, unless Python will not recognize the variable as a tuple. One item tuple, remember the commma:

    myTuple = ("SQL",)
print(type(myTuple))

    NOT a tuple

    myTuple = ("SQL")
print(type(myTuple))

    Joining Two Tuples

    To join two or more tuples you can use the + operator. Join two tuples:

    tuple1 = ("a", "b" , "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)

    The tuple() Constructor It is also possible to use the tuple() constructor to make a tuple. Using the tuple() method to make a tuple:

    myTuple = tuple(("SQL", "Python", "R")) # note the double round-brackets
print(myTuple)

    Tuple Methods

    Python has two built-in methods that you can use on tuples. count() Returns the number of times a specified value occurs in a tuple index() Searches the tuple for a specified value and returns the position of where it was found
