    Variables and Types

    Variables and Types

    Variables are reserved memory locations used to store values. Python has several basic data types that behave in ways that are similar to other programming languages. You do not need to declare variables before using them or declare their type. Every variable in Python is an object. Variables can hold any value and are case-sensitive. This tutorial will go over a few basic types of variables.

    Variable Names in Python

    There are a few rules governing the naming of variables:

    1. Variables can only contain letters, numbers, and underscores. Variable names can start with a letter or an underscore, but cannot start with a number.
    2. Reserved words or Python keywords cannot be used as variable names.
    3. There cannot be any spaces in a variable name. Spaces are not allowed in variable names, so we use underscores instead of spaces. For example, use car_wheel instead of “car wheel”.

    Variable Assignment

    The equals sign = is used to assign values to variables

    count = 100         # Assigning an integer value
name = "Pat"        # A string
measure = 100.0     # A floating point value

    Standard Data Types

    Python has five standard data types:

    • Number
    • String
    • List
    • Tuple
    • Dictionary
    • Set

    Python Numbers

    Python supports four different numerical types:

    int (signed integers)
long (long integers)
float (floating point numbers)
complex (complex numbers)

    To define an integer, the following syntax is used:

    myinteger = 7

    Floating point values are designated with a decimal point:

    myfloat = 7.0

    Complex numbers are made up of a real and an imaginary part. For example:

    myComplex = 7 + 2j

    Python Strings

    Strings are sequences of characters and are created by enclosing a sequence of characters in single or double quotes. Strings are immutable i.e. they cannot be changed once they’ve been created. They can, however, have characters or other strings added to them. Creating strings is straightforward:

    myString = 'Hello World!'

    Subsets of strings can be extracted using the slice operator []. String indices start at 0 to string length - 1.

    str[0]       # Selects first character of the string
str[2:5]     # Selects characters from the 3rd to the 5th
str[2:]      # Selects characters from the 3rd until the end    
str + "TEST" # Concatenates two strings

    Python Collections (Arrays)

    There are four collection data types in the Python programming language: List Tuple Set Dictionary

    Python Lists

    A list is a collection which is ordered, changeable and allows for duplicate members. Lists are created using square brackets.

    myList = ['apple', 'orange', 'pear']

    Python Tuples

    A tuple is a collection which is ordered and immutable i.e. it can’t be modified once it is assigned. Tuples are written using parentheses:

    myTuple = (1, 2, 3)

    Python Dictionaries

    Dictionaries are made up of a key and a value. They are created as follows:

    myDict = {'Tom':4, 'Pat': 4, 'John': 7}

    Python Sets

    A Set is a collection which is unordered and is mutable and iterable. It has no duplicate elements and follows the mathematical concept of a set. It is written as:

    mySet = set(["a", "b", "c"])
mySet = {"a", "b", "c"}

    Booleans

    Python implements all the usual operators associated with Boolean logic, but uses words rather than symbols

    t = True  # Assigning True to t
f = False # Assigning False to f    
t and f   # Logical AND
t or f    # Logical OR
not f     # Logical NOT
