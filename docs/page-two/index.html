    Variables and Types

    Variables and Types

    Variables are reserved memory locations used to store values. Python has several basic data types that behave in ways that are similar to other programming languages. You do not need to declare variables before using them or declare their type. Every variable in Python is an object. Variables can hold any value and are case-sensitive. This tutorial will go over a few basic types of variables.

    Variable Names

    There are a number of rules governing the naming of variables:

    1. Variable names cannot start with a number
    2. There can’t be any spaces in a variable name
    3. Reserved words cannot be used in variable names

    Variable Assignment

    The equals sign = is used to assign values to variables

    count = 100         # Assigning an integer value
name = "Pat"        # A string
measure = 100.0     # A floating point value

    Numbers

    Python supports four different numerical types:

    int (signed integers)
long (long integers)
float (floating point numbers)
complex (complex numbers)

    To define an integer, use the following syntax:

    myinteger = 7

    Booleans

    Python implements all the usual operators associated with Boolean logic, but uses words rather than symbols t= True f = False

    t and f # Logical AND
t or f # Logical OR
not f # Logical NOT

    Strings

    A string is a sequence of characters. Strings are created by enclosing a sequence of characters in single or double quotes. Strings are immutable i.e. they cannot be changed once they’ve been created. They can, however, have characters or other strings added to them.

    Subsets of strings can be extracted using the slice operator []. String indices start at 0 to string length - 1.

    str[0]       # Selects first character of the string
str[2:5]     # Selects characters from the 3rd to the 5th
str[2:]      # Selects characters from the 3rd until the end    
str + "TEST" # Concatenates two strings

    Lists

    Lists are created using square brackets.

    myList = ['apple', 'orange', 'pear']

    Tuples

    Tuples are created using parentheses.

    myTuple = (1, 2, 3)

    Dictionaries

    Dictionaries are made up of a key and a value. They are created as follows:

    myDict = {'Tom':4, 'Pat': 4, 'John': 7}
